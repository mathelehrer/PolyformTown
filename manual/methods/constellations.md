# Constellations

## Purpose

Enumerate and inspect vertex-completion constellations.

## Current command surface

- `./bin/vcomp_count ...`
- `./bin/vcomp_print ...`
- `./bin/vcomp_parity`

## Inputs

- Level/depth
- Tile file path
- Optional live filtering

## Outputs

- Count summaries
- Representative grouped records
- Parity checks for tracked/untracked variants
