# RL0 Data

RL0 data tracks base completion records and refinement/deletion state.

## Primary files

- `data/rl0/completions.dat`
- `data/rl0/deletions.dat`
- `data/rl0/completions.meta`

## Notes

- RL0 init path is currently the most trusted runlevel entry stage.
- Refine semantics are actively being clarified and documented.
