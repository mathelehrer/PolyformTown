# Data Trust

## Problem Statement

This program generates data to support a math proof. Working with 
deceitful code assistant gives pause, because faulty data can not 
support the validity of a theorem. We need 100% true data. 

What can we do to correct for drift and other common problems? 

--Require validation against prior knowledge libraries
--Reproduce key results by different methods
--Last (and best) resort: code auditing
